Transcribe y comparte sin fricción desde el primer minuto.

Grabadora convierte tus sesiones de audio en texto accionable en segundos. GPU workers procesan el audio, Redis coordina los avances y tú recibes deltas legibles al instante mientras guardamos cada resultado en tu almacenamiento S3.

Puesta en marcha en cuatro pasos

Configura variables

Duplica .env.example, completa tus credenciales S3/JWT y define la cuantización preferida.

Levanta el stack

Ejecuta docker compose up --build desde la raíz. El orquestador deja API, worker, Redis, DB, MinIO y dashboards listos.

Explora y prueba

Ingresa a /docs, crea un usuario, solicita token y sube audio en /transcribe.

Sigue el stream

Conéctate a /transcribe/{"{job_id}"} mediante SSE para recibir progresos en vivo y recuperar el transcript final.

¿Prefieres copiar y pegar?

Ejemplo rápido con HTTPX para crear un trabajo y escuchar el stream en CLI:

import httpx, sseclient

auth = httpx.post(
    "http://localhost:8000/auth/token",
    data={"username": "demo@grabadora.ai", "password": "demo"},
).json()

headers = {"Authorization": f"Bearer {auth['access_token']}"}
files = {"file": open("audio.wav", "rb")}

job = httpx.post("http://localhost:8000/transcribe", headers=headers, files=files).json()

for event in sseclient.SSEClient(f"http://localhost:8000/transcribe/{job['job_id']}", headers=headers):
    print(event.event, event.data)
