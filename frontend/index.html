Grabadora

Plataforma unificada para transcripciones en tiempo real con colas Redis, almacenamiento S3 y métricas listas para producción. Recibe deltas de texto al instante mientras la IA procesa tu audio.

Abrir API Docs Vista Redoc

Cómo verla en acción

  1. Levanta la plataforma con docker compose up --build.
  2. Abre http://localhost:8000/ para esta landing y /docs para la API interactiva.
  3. Crea un usuario, solicita un token y sube audio en /transcribe para seguir el SSE en /transcribe/<job_id>.

Qué incluye

  • Streaming SSE de tokens delta impulsado por faster-whisper con cuantización configurable.
  • Colas RQ sobre Redis con persistencia en MinIO/S3 de audios y transcripciones.
  • Métricas Prometheus y dashboards listos para Grafana en deploy/grafana/.
